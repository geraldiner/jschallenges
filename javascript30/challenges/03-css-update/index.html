Day 03

Updating CSS with JavaScript

Problem

Today's prompt is using JavaScript to update CSS variables in the DOM. In the example, you can use a slider to adjust the spacing of the image from the top-left corner, the blur, and the base color (wherever it may be referenced).

For this prompt, I decided to follow along with the video since I hadn't really worked with this before.

Solution

As always, I set up the HTML first. There is the title that shows what the page is all about, with the "JS" part in a different color to show the base color changing with the color picker.

The controls are simple different types of input. There is a "data-sizing" attribute on the Spacing and Blur inputs because we need those to update them with the JavaScript later.

Truthfully, I had no idea what :root was, and I'm still a bit confused on what it is. But after watching the video, my understanding is that it's like setting a global constant variable, which you can use in any other of the elements' css. Changing that one variable, changes all of them. So this is what we're doing with the spacing, blur, and color. That's why when the color is changed, the background of the image and the color of the JS part are changed at the same time.

With the event listeners on the input tags, we can access the value and name of the input to make sure we're changin the correct variable. For the Spacing and Blur values, we also need to add the 'px' suffix to ensure that the property gets read correctly.

Lessons Learned

I know that these are probably really simple "tricks" but learning them for the first time makes me think how magical web development can be. A lot of the time, my problems come from not knowing the syntax, but with a lot more practice I know I can get better at it.
