Problem

Today is day 2 of practicing Array methods, and these are methods I’d never really seen, let alone use before.

There are 4 exercises, using the some, every, find, and findIndex functions.

Solution

These functions were simple enough for me to look at the documentation and figure out. I even got to practice writing in arrow notation.

Some differences between my solutions and Wes’s is that he initialized a new Date object and used the getFullYear function to get the current year. I just used ‘2018’, but I Wes’s version is definitely better because the year will not always be 2018.

In the last exercise, he also used spread (…) again, which is very new to me. I’ll have to read up on the documentation for that. This was for exercise 4 where we needed to delete the comment with a certain id. Instead of just using splice on the original array, he created a new array using slice on the old array and spreading it into the new array. He says it’s popular in the Redux world, which I’ve heard of, but still don’t really know what it is. Even more reading and research!
