Day 09: Dev Tools Tricks

Day 09

Dev Tools Tricks

Summary

Today wasn’t really building anything, but going over different functions we could do in the console that can be viewed in the console in Google Chrome. I am most familiar with console.log, which is what I use to test my code all the time. I also learned console.table because Wes uses it a lot. He introduced interpolation, using %s; styling, using %c, console.warn, console.error, console.info, and console.assert. Most of them are pretty self-explanatory. console.assert is the only one that is still a little confusing. It’s for testing, and will only log the error message if the test is false. He also mentioned a few that I don’t think I’ll be using too much, like console.count and grouping console.log’s.
